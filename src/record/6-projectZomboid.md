# 僵毁锻造

## 1. 高级熔炉

需要的材料
1. 石工抹刀
2. 石砖
3. 粘土混凝土
4. 大风箱


## 2. 高级锻造炉
1. 石工抹刀
2. 石砖
3. 粘土混凝土
4. 大风箱
5. 木桶

## 3. 锻造升级
1级锻造，厨房刀刀片可以作为钢材再锻造厨房刀刀片
4级锻造，大匕首刀片做大匕首刀片 或者 钢片铁片做皮带扣，
5级锻造，可以用猎刀刀片做猎刀刀片
7级锻造，带着锻造杂志7可以做原本8级的简易版双手长剑的剑刃，双手长剑的剑刃做长剑剑刃

## 4. 锻造的配方
1. 研究钢条1/4得到短剑刃
2. 研究短剑刃得到长剑刃
