# oxygen肉食主义成就 缓螺

## 计算一个动物（缓螺）每周期产出卡路里

- 各类条件
  1. 水行星
  2. 模块大小为64，一个模块5只小动物
  3. 繁殖度为67%的缓螺，孵化度20%
  4. 抓一个野生的，开始驯化，年龄小于13岁，25-2-5-5=13（25岁老死，2周期用来下蛋，5周期用来孵化，5周期用来长大）
  5. 如果用孵化器，年龄小于17岁。25-2-1-5=17（25岁老死，2周期用来下蛋，1周期用来孵化，5周期用来长大）

## 分析图表

- 图表中，从第一周期开始驯化，到第6周期，得到了一个孵化的小动物，也就是缓螺幼崽。1000卡路里。
- 图表第二行，当第一只蛋，等待孵化的5周期中，第一只驯化的动物依旧每1.5周期下一个蛋，第二个蛋在7.5周期的时候孵化了，也就是缓螺幼崽。1000卡路里。
- 图表第三行，依旧隔1.5周期下蛋，依旧1，5周期有一个小动物孵化。
- 以此类推： 每隔1.5周期一个小动物孵化。 每1.5周期得到1000卡路里。

## 特殊周期：驯化的动物老死

- 因为考虑动物老死，需要立即补充成年的小动物，用来保证1.5周期下一个蛋。
- 所以第一只蛋，孵化后，不能杀死。

- 在第13周期后，第一只驯化的死掉，此时下的第一个蛋，刚好可以补充回畜舍，继续保持每1.5周期下一个蛋。
- 同时，在这个周期，老死的动物变成了肉，弥补了前面没有杀掉小动物。

## 结论

- 一个缓螺每周期能提供的卡路里为： 1000 / 1.5 = 666卡路里
- 要确保每个动物的第一个蛋，孵化后，不能立即杀死，变为食物。因为要补充

## 动物老死，第一次补充动物

- 13岁的老死，补充了一只6.5岁的。

| 畜舍中下蛋的动物年龄 | 用于补充的动物年龄 |
| -------------------- | ------------------ |
| 6.5岁被补充进畜舍    | -                  |
| 6.5+1.5=8岁          | 蛋                 |
| 8+5=13岁             | 孵化1岁            |
| 13+12=25岁           | 1+12=13岁          |

- 这只6.5岁的大儿子，1.5周期下第一个蛋的时候，大儿子8岁。
- 大儿子第一个蛋，孵化的时候，大儿子8+5=13岁。大孙子1岁。
- 大儿子老死25-13=12，大孙子此时1+12=13岁。补充回去继续保持下蛋。

## 动物老死，再次补充

| 畜舍中下蛋的动物年龄 | 用于补充的动物年龄 |
| -------------------- | ------------------ |
| 13岁被补充进畜舍     | -                  |
| 13+1.5=14.5岁        | 蛋                 |
| 14.5+5=19.5岁        | 孵化1岁            |
| 19.5+5.5=25岁        | 1+5.5=6.5岁        |

- 13岁的开始下蛋。
- 13+1.5=14.5岁，第一个蛋。
- 14.5+5=19.5岁，孵化幼体。
- 25-19.5=5.5周期老死，5.5岁刚成年。

- 大儿子第一个蛋，孵化的时候，大儿子8+5=13岁。大孙子1岁。
- 大儿子老死25-13=12，大孙子此时1+12=13岁。补充回去继续保持下蛋。

## 核心逻辑跳跃

- 我们将每一代接班时的年龄设为 x。
- 当一只成年动物（年龄 x）接班时，它需要 6.5 周期来准备下一代（1.5 周期产蛋 + 5 周期孵化）。
- 此时，它距离老死（25岁）剩余的寿命为 25 - x。
- 新一代在老一代死亡时的年龄，就是老一代剩余寿命减去繁育后代消耗的时间。由此直接得出最终变换公式：y = (25 - x) - 6.5

系统自稳演变

- 将你推演的数字直接代入该收敛公式：
- 初始干预：$A_1 = 14$ 岁
- 断档出现：$A_2 = 18.5 - 14 = 4.5$ 岁 （这是唯一一次断档，产生 0.5 周期的产能真空期）
- 首次自修：$A_3 = 18.5 - 5 = 13.5$ 岁 （因为 4.5 岁必须满 5 岁才启动繁育）
- 完美衔接：$A_4 = 18.5 - 13.5 = 5.0$ 岁 （老死瞬间，新一代刚好满 5 岁成年，无缝接班）
- 永久循环：$A_5 = 18.5 - 5.0 = 13.5$ 岁
- 系统在第三代之后彻底锁死在 13.5 岁 $\leftrightarrow$ 5.0 岁 的完美震荡中。
- 这 0.5 周期的未成年断档是单次损耗，在无限循环的挂机周期中，它对理论极值（666 卡路里/周期）的稀释作用会被无限抹平。
